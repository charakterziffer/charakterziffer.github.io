Nützliches aus meiner userContent.css

Mit der Datei userContent.css kann man Websites im Firefox nach seinen Bedürfnissen umgestalten, siehe mein Artikel über User-CSS. Einige meiner Modifikationen sind:

/* DIVERSE SEITEN: Silbentrennung aktivieren */
@-moz-document domain(medium.com), domain(wikipedia.org), domain(smashingmagazine.com), domain(deutschlandfunknova.de) {
body { hyphens: auto; }
h1, h2, h3, h4, h5, h6, code, pre, samp, kbd, var, abbr, acronym, sub, sup, textarea, label { hyphens: none !important; } }

/* FACEBOOK.COM: Einlogg-Aufforderungen verstecken,
fixierte Menüleiste mitscrollbar machen */
@-moz-document domain(facebook.com) {
.timelineLoggedOutSignUp, #pagelet_growth_expanding_cta, ._5hn6 {
	display:none; }
#headerArea, .fbPageBanner { display: none; }
.fixed_elem, ._k { position: relative !important; } }

/* PINTEREST.COM: Einlogg-Aufforderung ausblenden */
@-moz-document domain(pinterest.com), domain(pinterest.de) {
footer { display:none !important; } }

/* WIKIPEDIA.ORG: Quellenangaben mehrspaltig */
@-moz-document domain(wikipedia.org) {
.references { columns: 15em; column-gap: 3em;
	margin: 1em 0 0 1em !important; } }
